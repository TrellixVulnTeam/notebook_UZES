
引用: 变量的别名，用到的地方可以认为就是原变量，所以sizeof()测不出本质大小

```c++
int age = 10;
int &ref = age;
cout << sizeof(ref) << endl;
```

输出 4 (int的大小)

我们可以侧面测出ref的大小

```c++
struct sct {
    int &a;
};

cout << sizeof(sct) << endl;
```

输出8
